# Retail Sales Analysis Dashboard

## Project Overview
This project analyzes retail sales data using MongoDB, RapidMiner, and Power BI.

## Tools Used
- MongoDB Atlas / Compass
- RapidMiner / Altair AI Studio
- Power BI
- CSV dataset

## Project Workflow
CSV Dataset → MongoDB → RapidMiner Data Preparation → Power BI Dashboard

## Dashboard Features
- Total Revenue
- Total Profit
- Total Orders
- Total Customers
- Revenue by Product Category
- Top 10 Products by Revenue
- Orders by Payment Method
- Monthly Sales Trend

## Key Insights
- Electronics generated the highest revenue.
- Online Banking was the most used payment method.
- Sales increased strongly around March 2026.
- AlphaTech Keyboard was the top revenue-generating product.
